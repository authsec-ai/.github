## AuthSec is an Agent Native Authentication and Authorization platform

## What does AuthSec enable?

- enable SAML2 / OIDC SSO logins for your AI agents and MCP servers
- enable headerless authentication in your agents using Client Initiated Backchannel Authentication (CIBA)
- Enable RBAC for your AI agents
- Enable storing and retrieval of secrets.
